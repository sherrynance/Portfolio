Authentic Grace 2.0



As another assignment from Cohort 4 of the Moms Can: Code School we were to add few features to the Authentic Grace storefront. I added a product page that could be accessed by clicking the Shirts image under "Our Products" on the homepage. We were also tasked to create an accessibility report based on the homepage and to update the homepage to be more accessible.

Technologies used: HTML, CSS, JavaScript, Flexbox, Figma, Git, GitHub
